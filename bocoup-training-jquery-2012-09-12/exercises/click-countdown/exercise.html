These buttons should count down when clicked. (exercise)


Modify exercise.js to make the exercise page look like the solution page.


  • Prevent the form from being submitted!
  • Decrease the number when a button is clicked.
  • Allow each button to be clicked no more than the specified number of times.
  • Unbind the event handler and disable the button when the button should no longer be clickable.
  • Pluralize the word time accordingly: singular for 1, plural for 0 and other numbers.

If you need help, use the jQuery API documentation, MDN JavaScript Reference and the browser's developer tools!
