These lists need to be sorted. (exercise)


Modify exercise.js to make the exercise page look like the solution page.


  • Iterate over every list.
  • Get that list's listitems as an array of DOM elements, using the jQuery .get() method.
  • Sort the array, using the JavaScript Array .sort(compareFunction) method.
  • Append elements to the list to reflect the new sort order.
  • Take care not to unbind any bound event handlers.

If you need help, use the jQuery API documentation, MDN JavaScript Reference and the browser's developer tools!
