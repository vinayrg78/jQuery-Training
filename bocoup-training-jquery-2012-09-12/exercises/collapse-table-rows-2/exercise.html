Clicking a table header should collapse the rows after it. (exercise)


Modify exercise.js to make the exercise page look like the solution page.


  • The tables should behave appropriately whether or not JavaScript is enabled.
  • Utilize already-existing CSS rules to enable a hand cursor on table headers.
  • Add an icon to each table header using this HTML: <i class="icon-minus"></i>.
  • Wrap the inside of each table cell with a div element, since table cells and rows can't be animated.
  • When a table header is clicked,
    • any existing animation should be stopped.
    • the wrapper div elements should be animated with .slideToggle().
    • the tbody should be hidden in the animation callback function, so that extraneous borders aren't visible.
    • that header's icon should toggle between icon-minus and icon-plus appropriately.

If you need help, use the jQuery API documentation, MDN JavaScript Reference and the browser's developer tools!
