Each table column needs a “sum” row. (solution)


Modify exercise.js to make the exercise page look like the solution page.


  • Iterate over every table (as well as table row and table cell).
  • Store table cell sums in an array, indexed by the table cell's index.
  • Create an additional row at the end of each table containing the sums.
  • Give each sum cell a distinctive style, so it stands out from the other cells.

If you need help, use the jQuery API documentation, MDN JavaScript Reference and the browser's developer tools!
